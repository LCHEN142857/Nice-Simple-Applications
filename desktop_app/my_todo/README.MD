#  My Todo - 桌面待办事项清单  

一个Windows桌面待办小组件，固定在桌面上，具有黑灰渐变背景、背景透明度调节、历史待办管理等功能。

##  功能特性  

| 功能 | 描述 |
|------|------|
| 🖥️ **桌面小组件** | 固定在桌面，不置顶，像原生桌面组件一样 |
| 🎨 **黑灰渐变背景** | 从黑色到灰色的渐变背景，现代简洁 |
| 🔍 **背景透明度** | 仅背景透明，文字始终清晰可见 |
| 📜 **历史待办** | 完成的待办自动归档到历史记录 |
| 🧲 **桌面吸附** | 拖动窗口到屏幕边缘时自动吸附 |
| 💾 **数据持久化** | 使用 SQLite 本地数据库存储 |
